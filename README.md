# GitHub Profile Finder

A modern and responsive web application that allows users to search for any GitHub profile and view detailed information in real time using the GitHub REST API.

The application displays essential profile details such as the user's avatar, name, bio, location, followers, following, public repositories, and profile link. It also showcases the user's latest repositories with useful information like stars, forks, and primary programming language.

## ✨ Features

* 🔍 Search for any GitHub user by username
* 👤 View profile information in real time
* 📚 Display public repositories
* ⭐ Show repository stars and forks
* 📍 View bio, location, and account statistics
* 📱 Fully responsive design for desktop and mobile
* ⚡ Fast and clean user interface
* 🌐 Powered by the GitHub REST API

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (ES6+)
* GitHub REST API

## 🚀 Getting Started

1. Clone this repository.
2. Open the project folder.
3. Run the project using Live Server or open `index.html` in your browser.
4. Search for any GitHub username to explore profile details.

## 📖 How It Works

The application sends a request to the GitHub REST API whenever a username is searched. It then fetches the user's public profile data and repositories, displaying the information in a clean and user-friendly interface.

## 🎯 Purpose

This project was built to practice working with APIs, asynchronous JavaScript (`fetch`), and responsive frontend development while creating a useful real-world application.
